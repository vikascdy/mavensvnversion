To add custom scroll bar for Sencha Components like Tree & Grid. Add below files in the head section of HTML File. Custom scroll bar will render automatically for Tree & Grid.

  • Latest jQuery lib
    • <script type="text/javascript" src="../jquery/jquery-1.10.1.min.js"></script>
  • jScrollPane
    • <script type="text/javascript" src="../packages/ext-theme-edifecs/build/resources/js/JScrollPane.js"></script>
  • Edifecs theme JS
    • <script type="text/javascript" src="../packages/ext-theme-edifecs/build/ext-theme-edifecs.js"></script>

To add custom scroll bar for Sencha Component like Panel. Add above files in the head section of HTML File and call setPanelScroller(me) function with this argument in afterRender listener for Panel. Custom Scroll Bar did not work for nested panels.
