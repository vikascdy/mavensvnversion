Following are the events which are exposed by Application Bar to handle Navigation, Notification and Favorites menu items in a controller.

  • Task Event
    • 'taskItemClicked'
  • Menu Item Click Event
    • 'menuItemClicked'
  • Show All Notification Clicked Event
    • 'showAllNotificationClicked'
  • Favourite Item Clicked Event
    • 'favouriteItemClicked'
  • Search Item Clicked Event
    • 'searchClicked'
