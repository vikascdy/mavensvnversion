Application Development Setup

  • Download Latest Shared UI Framework Bundle. (Latest Shared UI Bundle 3.0-preview)
  • Extract zip file of Shared UI Bundle and treat as a Workspace.
  • Download Sencha CMD 3.1.2.342. Install below prerequisite software for Sencha CMD
    • Install JDK1.6 or 1.7 (Java Development Kit)
    • Install Ruby 1.9.3 (or earlier): Download Ruby 1.9.3.n from rubyinstaller.org
  • Open command prompt & go to Workspace as mentioned above.
  • Run "sencha" command and verify Sencha CMD installed or not.
  • From Workspace directory create Application with following command
    • "sencha -sdk ext generate app Analysis-Set analysis-set"
  • Analysis-Set application folder is created in Workspace with above command. Go to Analysis-Set application folder.
  • Change the path in bootstrap.css file so that it refer to edifecs-theme css i.e.
    • Change @import '../packages/ext-theme-classic/build/resources/ext-theme-classic-all.css' to @import '../packages/ext-theme-edifecs/build/resources/ext-theme-edifecs-all.css'
  • Under Application (analysis-set) go to .sencha/app/sencha.cfg file and update app.classpath variable so that it can point to ext-theme-edifecs i.e.
    • app.theme=ext-theme-edifecs

Application Build Process

  • Under Application (analysis-set) go to .sencha/app/sencha.cfg file and update app.classpath variable so that it can point to edifecs-plugins i.e.
    • app.classpath=${app.dir}/app, ${app.dir}/app.js, ${workspace.dir}/edifecs-plugins
  • If Application (analysis-set) uses any edifecs custom component from edifecs-plugins folder in Workspace then add manifest for edifecs custom components under Application (analysis-set) folder in sass/example/custom.js file e.g. 
    •                     Ext.theme.addManifest (
                        {
                            xtype: 'button',
                            ui: 'edifecs-applicationbar'
                        },
                        {
                            xtype: 'widget.button',
                            ui: 'edifecs-applicationbar-mousedown'
                        },
                        {
                            xtype: 'button',
                            ui: 'edifecs-applicationbar-submenu'
                        },
                        {
                            xtype: 'panel',
                            ui: 'edifecs-leftmenu'
                        }
                    );
  • Go to Application (analysis-set) from command prompt and run following command.
    • "sencha app build" (Production build) OR
    • "sencha app build testing" (Testing build without minified js/css)

Remove the Extra things from Final Build

  • Remove "config.rb" file from build/production/application folder
  • Remove "application-all.scss" file from build/production/application folder


Note:- Please let us know if you face any problem while implementing build process.
